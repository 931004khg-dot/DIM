## 1️⃣ 선(Line) 탭 설정

### 치수선

| 항목 | 설정값 | AutoCAD 변수 | 설명 |
|------|--------|--------------|------|
| 색상 | ByLayer | DIMCLRD = 256 | 레이어 색상 따름 |
| 선종류 | CONTIN | - | 연속선 (기본값) |
| 선가중치 | ByLayer | - | 레이어 선가중치 따름 |
| 눈금 너머로 연장 | 0 | DIMDLE = 0.0 | 치수선이 화살표를 넘어 연장되지 않음 |
| 기준선 간격 | 3.75 | DIMDLI = 3.75 | 연속 치수 사이의 간격 |

## 🎯 ISO-25 표준의 특징

이 ISO-25 치수 스타일은 다음과 같은 특징을 가집니다:

### ✅ 주요 특징

1. **축척 20 적용**
   - 도면 축척 1:20에 최적화
   - 모든 치수 요소가 비율에 맞게 조정됨

2. **닫힌 채움 화살표**
   - ISO 표준의 전형적인 화살표 스타일
   - 모든 화살표가 동일한 스타일 사용

3. **문자 높이 3**
   - 출력 시 적절한 가독성 확보
   - A1, A2 사이즈 도면에 적합

4. **십진법 단위**
   - 소수점 둘째 자리까지 표시 (0.00)
   - 선행 0 억제 (0.5 → .5)

5. **ByLayer 설정**
   - 레이어별로 색상/선종류 제어 가능
   - 도면 관리 용이

### 📐 적용 범위

- **도면 축척**: 1:20 (DIMSCALE = 20)
- **용지 크기**: A1, A2, A3
- **단위**: mm (밀리미터)
- **정밀도**: 소수점 2자리

### 🔧 커스터마이징

DCL 대화상자에서 다음 항목을 조정할 수 있습니다:

- 전체 축척 (DIMSCALE): 도면 축척에 맞게 조정
- 문자 높이 (DIMTXT): 출력 크기에 맞게 조정
- 화살표 크기 (DIMASZ): 선호도에 따라 조정
- 치수보조선 간격 (DIMEXO): 객체와의 간격 조정
- 치수보조선 연장 (DIMEXE): 연장 길이 조정
- 문자 간격 (DIMGAP): 문자와 치수선 간격 조정
